--- layout: post title: Add missing file extension to file names tags: - Asides - Lifehack - Aside status: publish type: post published: true meta: _edit_last: '1' --- For Mac users: if you ever happen to download a bunch of similar files and they're all missing extensions (like a directory full of photos crawled using wget), here's a quick one-line command to add the extension to all the files: for i in * ; do mv $i $i.txt; done;
