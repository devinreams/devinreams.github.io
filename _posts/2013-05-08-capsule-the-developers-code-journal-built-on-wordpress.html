Capsule replaces that scratch document you have open when you're coding. It creates an archive of your development artifacts.
Instead of keeping a text file open when working on a project, using Capsule means you can have a simple archive of all those notes and easily reference them in the future. Initial reactions and reception have been very positive from the development and WordPress community so we're all very pleased. Be sure to check out Alex's post on Capsule to read more about the thinking and decisions behind this (free) product.
